# Three-Day Marketability Pass

Date: 2026-06-25

## Practical Differentiator

Project N.O.M.A.D. is strongest as a free Linux/Docker offline server with optional AI, education, maps, and content management. Offgrid Kit should avoid competing as "another server stack" and instead lean into a paid-product advantage:

- plug-in USB handoff
- Windows/macOS/Linux launch flow
- bundled content instead of mandatory first-install downloads
- non-technical dashboard
- printable field support for users who may not be comfortable troubleshooting services

## Added For This Pass

- `/field-cards`: printable pocket-style cards for starting the kit, power outage use, water search, first aid search, maps, translation/Morse, AI prompting, and installed-content proof.
- `/offline-proof`: customer-facing proof page with live installed counts, local service URLs, privacy posture, and support status.

## Why This Helps

N.O.M.A.D. can look stronger to technical buyers because it has Docker orchestration, Kolibri, CyberChef, and Qdrant/Ollama integrations. Offgrid Kit can look stronger to mainstream paid buyers by being easier to launch, easier to hand to a family member, and easier to trust during an outage.

The product message should be:

> Project N.O.M.A.D. is a server you install. Offgrid Kit is a ready-to-use offline drive you can hand to someone.

## Still Worth Considering After Tester Launch

- Optional education pack: Kolibri or static Khan/CK-12 content.
- Optional utility pack: CyberChef or equivalent static tools.
- Semantic vector search for documents/ZIM extracts when storage allows.
- In-app content pack manager for public-domain packs.
